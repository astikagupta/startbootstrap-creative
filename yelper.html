YELPER : An app that brings together the best of Yelp and Uber worlds



User Interface Design | Prototyping | Software Development in Python


My Role

Lead efforts in creating the user interface of the app and software development in Python.

Summary

Yelper aims to simplify the process of dining at restaurants. The application accepts the user’s input address, (the address he/she) wishes to be picked up from. Once the address is received, it is sent to the Google API to convert the plain text address to latitude and longitude. At the same time the address is also sent to the Yelp API to retrieve the list of top 20 rated restaurants. Once the restaurants have been retrieved the addresses, along with the starting locations are sent to the Uber API to retrieve the distances and the prices to travel to each restaurant and are then displayed on the screen in the descending order of ratings. The user can then pick a restaurant based on the best possible option.

Features

∙ Allows the user to enter any starting address in plain text.
∙ Allows the user to enter the destination address where he wants to find a restaurant in plain text.
∙ Displays a list of top rated restaurants from Yelp in and around the address mentioned by the user.
∙ Displays the cost of travel, time of travel and distance to be covered by Uber.
∙ Uses the google API, Uber API and Yelp API to provide the relevant results.

Technologies Used

Apart from HTML, CSS, and Javascript, we also used the Jinja framework to iterate through the list of results that we fetched using the APIs to render them on the results page. We used the Python Flask server to host our application.
